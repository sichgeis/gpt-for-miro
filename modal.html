open GPT for Miro

Bring Chat-GPT to your Miro board.

Select stickies and give an instruction what to do with them. GPT for Miro then replies with stickies that you can directly use on your board.

Check out a demo video here!

Let's go!
 

To obtain an OpenAI API key, head over to platform.openai.com and sign up for an account, if you don't already have one. Navigate to the API keys section to create a new key. Return here and insert the API key in the field above to start using GPT for Miro.

The App stores the api key exclusively in your browser. Your api key is not shared with anyone.
